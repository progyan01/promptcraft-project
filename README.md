# NeoFrisco

### PromptCraft Hackathon - Track 2

This is a 100% AI-generated "digital artifact" created for the PromptCraft Hackathon.
It is an interactive, map-based interface for the fictional city of NeoFrisco.

### Core Features

* **100% AI-Generated:** All HTML, CSS, and JavaScript were generated in a single file by an AI (Google's Gemini).

* **Interactive Map:** The city image serves as the main navigation. Users can discover and hover over invisible, glitching hotspots to open information panels.

* **Dynamic UI:** The interface is filled with moving elements to create an immersive "vibe".

### How to Run This Artifact

This is a self-contained digital artifact. No servers, builds, or dependencies are required.

1.  Download the `index.html` file.

2.  **Double-click the `index.html` file.**

3.  It will open directly in any modern web browser (Chrome, Firefox, Edge, Safari).
